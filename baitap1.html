What is CSS

Cascading style sheet (CSS) is used to format the layout of a webage
With CSS, you can control, font, the size of the text, the spacing between elements, how elements are positioned and laid out , what background colors are used, differents devices and screen sizes, and much more!
Tip: the word cascading means that a style applied a parent element will also apply to all children elements will also apply to all children elements within the parent. So, if you set the color of the body text to "blue", all headings, paragraph, and other text elements within the body will also get the same color (unless you specify something else)
